# Base del Proyecto Backend (Arquetipos)

## Descripción
Estructura base utilizada para todos los microservicios del proyecto.

## Uso
Para generar un nuevo microservicio siguiendo este estándar:
1. Copiar la estructura de una carpeta de microservicio existente.
2. Actualizar el `artifactId` y `name` en el archivo `pom.xml`.
3. Configurar las variables de entorno en el `docker-compose.yml` centralizado.
